생각보다 아우디 딜러를 찾기는 어려웠다.

​

의외로 주변에 벤츠, bmw 타는 사람은 많았지만 아우디를 타는 사람이 없었다. 오히려 테슬라를 타는 사람이 더 많았다. 아마도 디젤게이트 이후 이미지를 많이 깍아먹어서 인 것 같다. 

​

우선 전기차 카페의 아우디 출고 후기를 올리신 분들 위주로 딜러 소개를 부탁하는 쪽지를 드렸다. 쪽지는 바로 오지 않기에 선영이에게 흥덕맘카페와 향기맘에 혹시 있나 찾아봐 달라 했다. 향기맘은 왠지 있을 것 같았다.

​

그렇게 몇 명의 딜러 연락처를 알게 되었고, 다 연락을 했다. 이미 기본적인 프로모션에 대한 정보는 알고 있었기에 그 이하를 얘기하는 딜러는 건너뛰었다. 최종적으로 흥덕맘카페에서 소개 받은 동탄의 외본 소속 딜러분과 카페분이 쪽지로 알려준 일산의 태안모터스 소속 딜러분과 계약을 했다. 내가 사려고 했던 맨하탄그레이는 당장 재고가 없다 하여 양쪽에 계약금을 걸고 빠르게 나오는 쪽과 계약하겠다고 양쪽 딜러분에게 말씀드렸다.

​

개인적으로 기존 자동차 업계의 딜러를 통한 차량 판매는 구매자를 너무 귀찮게 하는 시스템이다. 기본 컨셉이 예전 용산에서 "얼마까지 알아보셨어요?" 와 같다. 같은 차를 사도 딜러마다 구매가격과 제공해주는 +@ 가 모두 다르다. 발품을 팔아야만 싸게 살 수 있다. 그리고 기본으로 제공해주신다는 것도 명쾌하게 바로 알려주는 딜러분은 거의 못 본 것 같다. 블랙박스, 틴팅 제공이라고만 하지 어떤 블랙박스를 어떤 틴팅을 제공한다고 먼저 알려주지 않는다.

​

그러나 난 발품팔며 버리는 시간보다 적당한 선에서 끝내는 걸 좋아한다. 어렸을 때는 엄청 발품을 팔아 최저가로 구매하면 매우 뿌듯하였으나, 나이가 찰 수록 "시간은 금이라구 친구" 라 말한 가젯잔 고블린의 말이 이해가 되었다. 엄청난 시간을 들여 최저가를 살 수 있겠지만 알려진 프로모션에 약간의 +@ 면 되는 것 같다. 

​

아우디를 처음 알아본 나에게는 e-tron의 이번 프로모션은 정말 좋아 보였다. 다들 통수의 아우디라며 더 좋은 프로모션이 나올 것이라는 글/댓글이 있었지만 사기로 할 때 사서 바로 사서 쓰는 게 나에겐 더 의미있었다. 나중에, 나중에 하다가는 살 수 있는 물건이 없는 것이다.

​

이미 널리 알려진 프로모션이지만, 정리하면

| 항목 | 금액 |
| ----| ---- |
| 아우디 파이낸셜 최소 4천 사용 | 9,100만 |
| 트레이드인 | -220만 |
| 딜러 페이백 | -300만 |
| 세금 제외 실 구매가 | 8,580만 |

여기에 +@ 로

| 항목 | 금액 | 비고 |
| ----| ---- | ---- |
| 아우디 블랙 카드 <br> 또는 <br>완속충전기 설치 | 200만원 | 아우디 또는 대영채비 충전기에서 사용 가능<br> 또는 <br>대영채비 업체에서 집에 완속충전기를 설치 |
| 아우디 흰색 카드 | 100만원 | 아우디 또는 대영채비 충전기에서 사용 가능 |
| 딜러5종 | | 하이패스<br>블랙박스<br>틴팅<br>ppf<br>유리막 |

다들 딜러 5종 대신 돈으로 받고 사설업체를 쓰는 게 더 좋다고 하는데, 난 그렇게 시간 버리는 게 너무 싫었다. 틴팅, 블랙박스를 잠깐 알아봤었는데 제품이 너무 많았다. 제품을 알아보고 비교하고 그러는 데 시간 버리고 싶지 않았다. 딜러표라고 형편없는 것은 아니고 그저 가성비 좋은 저렴한 버전이며 고가의 제품과 큰 차이가 난다고 생각하진 않는다. 다만, 기본으로 해주시는 것에 3가지 요청을 하였다.

​

- 제공해주는 블랙박스는 사용기가 없을 정도로 잘 안 알려진 제품이어서 파인뷰로 변경

- 틴팅은 glc350e 때 금속성으로 했더니 우리 동네 차단기 인식이 안 되길래 무조건 비-금속성 틴팅

- 그리고 틴팅은 전면은 무조건 밝은 색으로

​

결론적으로 일산의 딜러분이 차를 먼저 구해오셔서 일산의 딜러분과 계약을 계속 진행하였다. 흥덕맘카페에서 소개받은 딜러분도 친절하시고 좋았는데, 아쉽지만 계약금 환불을 요청하였다.
